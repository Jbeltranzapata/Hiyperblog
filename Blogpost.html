practicando lo aprendido en las clases 15 y 16 de Platzi, se realiza practica de los siguientes codigos;

Git init -> es para iniciar inicar la carpeta de trabajo en el directorio de trabajo
Git add -> agrega todos los repositorios al repositorio local
Git commit -m "mensaje" -> guarda los cambios del archivo en
Git push -> introduce las actualizaciones en el repositorio remoto
Git pull -> obtiene las actualizaciones del repositorio remoto
Git git commit -am -> guarda todas las actualizacion realizadas en el directorio de trabajo
Git status -> muestra las actualizaciones realizadas y pendientes por guardar
Git merge -> combina las actualizaciones realizas en otra rama con el main
Git branch -> crea una rama nueva
Git checkout -> permite moverse entre ramas
